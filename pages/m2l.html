Projet 2

M2L Projet WEB

Contexte :

Dans le cadre de ma formation en BTS SIO, j'ai réalisé un projet Web.
Ce projet consistait à créer un site web pour la Maison des Ligues de Lorraine. Ce site web devait permettre aux utilisateurs d'acceder à un annuaire de collaborateurs, de pouvoir modifier ces informations via
une page profil et de pouvoir se connecter et se deconnecter.

Compétences acquises :

Ce projet m'a permis d'acquérir des compétences en développement web et en gestion de projet. J'ai pu apprendre à créer un site web responsive.

Compétences acquise :

- Traiter des demandes concernant les applications
- Analyser les objectifs et les modalités d’organisation d’un projet
- Déployer un service

Objectifs :

Les objectifs de ce projet étaient de créer un site web en utilisant les langages HTML, CSS, PHP et SQL

Environnement :

- Visual Studio Code
- Git
- GitHub
